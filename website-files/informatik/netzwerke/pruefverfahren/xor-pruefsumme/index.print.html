XOR-Pruefsumme

Definition:
Mehrere gleich lange Teile einer Bitfolge werden XOR-verknüpft. Es entsteht eine Prüfsumme, die die gleiche Länge wie die einzelnen Bitfolgen hat.

	Beispiel:
	Bitfolge: 10110011
	Berechnung:		10110011  
			XOR	01101000
	Prüfsumme:		11011011
